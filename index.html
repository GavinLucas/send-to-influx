send-to-influx APT repository (frozen)

This project has moved to L337-org/send-to-influx. This URL serves a frozen snapshot (last version: 4.2) so existing installs keep working - it will never receive updates. The live APT repository is at apt.l337.org.

To switch an existing install to the new URL:

sudo sed -i 's|https://gavinlucas.github.io/send-to-influx/|https://apt.l337.org/|' /etc/apt/sources.list.d/send-to-influx.list
curl -fsSL https://apt.l337.org/send-to-influx.gpg | sudo tee /usr/share/keyrings/send-to-influx.gpg >/dev/null
sudo apt update

For a fresh install, follow the README at the new home.
